# 1inch SDK

> ⚠️ Beta ⚠️
Might be unstable at this moment

`1inch-sdk` is a JavaScript library that  provide wrapper around [1inch Aggregator API v3](https://docs.1inch.io/api/).

✅ Works with this blockchains:
  - Ethereum
  - Binance Smart Chain
  - Polygon
  - Optimism
